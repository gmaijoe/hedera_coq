Swirlds Hashgraph Consensus Algorithm

The core components of the formalism are defined in the following files:
Swirlds.HashGraph
The definition of a hashgraph.
Swirlds.Definitions
The core hashgraph definitions including: ancestor, selfAncestor, manyCreators
Swirlds.See
The definition of see and stronglySee as well as the strongly seeing lemma (lemma 5.12).
Swirlds.Round
The definition of round and a proof that arounds are consistent (lemma 5.13).
Swirlds.Voting
The definition of vote and related voting definitions.
Swirlds.Swirlds
The proofs about the consense algorithm.
